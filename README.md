# Recipes

A personal cookbook organized by course and protein, formatted for [Obsidian](https://obsidian.md) with YAML frontmatter for tagging, search, and graph linking.

## Structure

```
appetizers/
  salads/
  soups/
entrees/
  fish/ shellfish/ poultry/ beef/ pork/ lamb/ vegetarian/
desserts/
sides/
```

Each recipe is a `.md` file with frontmatter tags covering course, protein, cuisine, technique, difficulty, time, season, dietary restrictions, and flavour profile.

## Adding Recipes

See `CLAUDE.md` for the full recipe format spec, tag reference, and writing standards.
